# Web UI

Локальный интерфейс оператора для рабочего backend (`Collector --serve`).

## Структура
- `index.html` — разметка экранов.
- `styles.css` — общие стили и состояния интерфейса.
- `uikit.js` — утилиты UI (toast, notice, делегирование событий).
- `app.js` — клиентская логика, рендер, работа с backend API.

## Основные сценарии
- Актуализация базы клиентов из Rocketman.
- Формирование плановой очереди по фильтрам (TZ/просрочка/точный день).
- Запуск и остановка live-рассылки.
- Управление Android-каналами (добавление, редактирование, проверка, включение/отключение).
- Ручное редактирование очереди, назначение шаблонов и переотправка ошибок.
- Диалоги и ручные исходящие сообщения.
- Стоп-лист и отчеты.
- Карточки уведомлений об ошибках и их статусах.

## Примечания
- Источник правды для UI — только backend API и БД.
- Функции автоматической синхронизации входящих SMS с устройств отключены.
